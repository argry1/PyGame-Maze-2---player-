# Лабиринт - Игра на Pygame

Увлекательная игра-лабиринт с двумя игроками, тремя уровнями сложности и системой бонусов. Играйте за фиолетового и синего персонажа, собирайте призы и проходите лабиринты!

## Возможности

- **Два игрока**: Управляйте двумя персонажами одновременно
- **Три уровня**: Различные лабиринты с возрастающей сложностью
- **Система бонусов**: Собирайте призы для победы
- **Звуковое сопровождение**: Фоновая музыка и звуковые эффекты
- **Выбор уровня**: Переключайтесь между уровнями нажатием клавиш 1, 2, 3
- **Два режима победы**: Каждый игрок может выиграть отдельно

## Управление

### Первый игрок (Фиолетовый квадрат)
- **Стрелки** (← ↑ ↓ →) - перемещение

### Второй игрок (Синий квадрат)
- **W A S D** - перемещение

### Дополнительное управление
- **F H T G** - управление дополнительным объектом (фиолетовый призрак)
- **1, 2, 3** - выбор уровня на экране выбора
- **Любая клавиша/мышь** - переход между экранами
